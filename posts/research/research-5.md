# [One Model to Edit Them All:Free-Form Text-Driven Image Manipulation with Semantic Modulations](https://arxiv.org/abs/2210.07883)

## Abstract

We propose FFCLIP to support Free-form text prompts for image manipulation. We are the **first** method to achieve all semantic editing using a single model.

## Methodology
- Semantic Alignment Module. The semantic alignment performs the automatic latent mapping via linear transformations with a cross attention mechanism.
- Semantic Injection Module. We inject semantics from text prompt embeddings to the StyleGAN latent space.

# [一统天下的模型:基于语义调制支持自由文本输入的图像编辑](https://arxiv.org/abs/2210.07883)

## 摘要
我们提出 FFCLIP，旨在支持图像操作的自由文本提示。我们是**首个**使用单一模型实现所有语义编辑的方法。

## 方法论
- 语义对齐模块。语义对齐模块通过线性变换和交叉注意力机制实现自动潜在映射。

- 语义注入模块。我们将文本提示嵌入中的语义信息注入到 StyleGAN 的潜在空间中。

# [Weakly Supervised Instance Segmentation By Exploring Entire Object Regions](https://ieeexplore.ieee.org/abstract/document/9612036)

## Abstract

We propose a novel self-supervised joint learning framework and consider the clustering problem as calculating the probability that pixels belong to each instance.

## Methodology
- In the first stream, we leverage a joint learning framework to implement image-level supervised semantic segmentation with self-supervised saliency detection.
- In the second stream, we propose a Center Detection Network to detect different instances’ centers with the gaussian loss function to cluster instances pixels. 
- Besides, an integration module is utilized to combine information of both streams and get precise pseudo instances labels.

# [通过探索整个对象区域进行弱监督实例分割](https://ieeexplore.ieee.org/abstract/document/9612036)

## 摘要
我们提出了一种新颖的自监督联合学习框架，并将聚类问题视为计算像素属于每个实例的概率。

## 方法论
- 在第一个流程中，我们利用联合学习框架实现图像级监督语义分割和自监督显著性检测。

- 在第二个流程中，我们提出了一种中心检测网络，利用高斯损失函数检测不同实例的中心，从而对实例像素进行聚类。

- 此外，我们还使用了一个集成模块来融合两个流程的信息，以获得精确的伪实例标签。
